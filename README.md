
# Toasties Sunday

>A modern web app for managing your Sunday toasty event.

## Features

- **Menu**: Browse and select from a variety of toasties.
- **Who's Coming**: See and manage the guest list.
- **Payment**: Pay easily via PayPal or Stripe.
- **Order Collection**: Organize and track orders efficiently.
- **Account**: Manage your personal account and preferences.
- **Feedback**: Give feedback on our cooking.
